# XDM 7.2.11 Chrome Extension

A Chrome extension for XDM (Xtreme Download Manager) version 7.2.11 that integrates with the latest Chrome browser using Manifest V3.

## Features

- Seamless integration with XDM download manager
- Support for the latest Chrome browser versions
- Built on Manifest V3 for enhanced security and performance
- Automatic download capture and routing to XDM
- Enhanced download management capabilities

## Compatibility

- **Chrome Browser**: Latest versions (Manifest V3 compatible)
- **XDM Version**: 7.2.11
- **Manifest Version**: V3

## Installation

1. Download the extension files
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension directory
5. Ensure XDM 7.2.11 is installed and running on your system
